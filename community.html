Main Code Repository

Project source code is stored in an Apache hosted git repository.

  • Repository: https://git-wip-us.apache.org/repos/asf/libcloud.git
  • Gitweb: https://git-wip-us.apache.org/repos/asf?p=libcloud.git

Website Code Repository

Website source code is stored in an Apache hosted svn repository. Source code is stored in an svn and not git repository, because svn-based Svnpubsub mechanism is used for distributing change notification to the web servers.

  • Repository: https://svn.apache.org/repos/asf/libcloud/site/trunk
  • ViewVc: https://svn.apache.org/viewvc/libcloud/site/
