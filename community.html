Community

Mailing Lists

  • announce@libcloud.apache.org (archive) - Only important project updates and release announcements are sent to this list (low volume).
  • users@libcloud.apache.org (archive) - Mailing list for Libcloud users (low volume)
  • dev@libcloud.apache.org (archive) - Mailing list for Libcloud developers and people interested in contributing to Libcloud (moderate volume)
  • commits@libcloud.apache.org (archive) - Commits messages, pull request notifications, build notifications and other automated messages go to this mailing list (high volume)

IRC Chat

  • #libcloud on Freenode

Issue Tracker

  • Apache JIRA: https://issues.apache.org/jira/browse/LIBCLOUD

Main Code Repository

Project source code is stored in an Apache hosted git repository.

  • Repository: https://git-wip-us.apache.org/repos/asf/libcloud.git
  • Gitweb: https://git-wip-us.apache.org/repos/asf?p=libcloud.git

Website Code Repository

Website source code is stored in an Apache hosted svn repository. Source code is stored in an svn and not git repository, because svn-based Svnpubsub mechanism is used for distributing change notification to the web servers.

  • Repository: https://svn.apache.org/repos/asf/libcloud/site/trunk
  • ViewVc: https://svn.apache.org/viewvc/libcloud/site/
